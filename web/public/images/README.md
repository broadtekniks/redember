# Red Ember Product Images

This directory contains placeholder SVG images for the Red Ember chili oil products.

## Available Images

- **placeholder-chili-oil.svg** - Generic chili oil bottle placeholder
- **product-original.svg** - Original Red Ember Chili Oil
- **product-spicy.svg** - Extra Spicy variant
- **product-garlic.svg** - Garlic Infusion variant
- **product-szechuan.svg** - Szechuan Peppercorn variant

## Usage in Components

```tsx
<img src="/images/product-original.svg" alt="Original Red Ember Chili Oil" />
```

## Image Specifications

- **Format:** SVG (scalable vector graphics)
- **Dimensions:** 400x400px viewBox
- **Colors:** Brand colors (#ec131e, #1b0d0e, #9a4c50, etc.)
- **File Size:** ~2-3KB each

## Customization

These are SVG files that can be edited in any text editor or vector graphics software like:

- Figma
- Adobe Illustrator
- Inkscape (free)
